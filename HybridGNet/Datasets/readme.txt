Download the datasets from the official websites, as we only provide the landmark annotations for JSRT.

JSRT: http://db.jsrt.or.jp/eng.php

Shenzhen and Montgomery: https://lhncbc.nlm.nih.gov/LHC-publications/pubs/TuberculosisChestXrayImageDataSets.html
Alternative link: https://openi.nlm.nih.gov/faq#faq-tb-coll

Shenzhen annotations: https://www.kaggle.com/yoctoman/shcxr-lung-mask

Save the data on the corresponding folders and run the preprocessing script for each dataset.
